val reify_type : 'a -> 'b t * 'b
given any value it returns its type and the same value. Note that since 'b t is GADT, the type system will reify its type automatically, for example 
    match reify_type "3" with
    | String, v -> v  ^ " this type safe control flow analysis will infer v as string"
    | _ -> assert false
